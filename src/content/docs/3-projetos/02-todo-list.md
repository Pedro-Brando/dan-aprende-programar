## O que você vai construir

Um app CLI de lista de tarefas: adicionar, listar pendentes/concluídas, marcar como feito, remover. Os dados ficam salvos em um `tarefas.json` para sobreviver entre execuções.

## Habilidades exercitadas

- Listas e dicionários compondo dados estruturados
- Leitura/escrita de arquivos
- Módulo `json` para persistência
- Funções dedicadas para cada ação (add, list, done, remove)
- Menu de comandos no terminal
