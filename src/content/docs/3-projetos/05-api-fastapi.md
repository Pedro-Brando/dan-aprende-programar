## O que você vai construir

Uma API REST com endpoints CRUD (criar/ler/atualizar/deletar) usando FastAPI, persistência em SQLite, validação com Pydantic e documentação automática em `/docs`.

## Habilidades exercitadas

- FastAPI: rotas, métodos HTTP, dependency injection básica
- Pydantic: schemas de request/response com validação
- SQLite via SQLAlchemy ou queries diretas
- Tratamento de erros com `HTTPException`
- Documentação automática (Swagger UI) - chamar atenção em entrevista
- Testes da API com `pytest` + `TestClient`

## Entrega esperada

Repo com README detalhado, exemplos de curl, screenshot do Swagger UI e ao menos 5 testes passando. Este é o projeto-âncora do portfólio.
