## O que você vai construir

Uma calculadora de terminal que aceita expressões como `12 + 7` e devolve o resultado, em loop até o usuário digitar `sair`.

## Habilidades exercitadas

- `input` / `print` / f-strings
- Condicionais e loops
- Funções com retorno
- Tratamento de erros (divisão por zero, entrada inválida)
- Estrutura de um arquivo `.py` executável

## Entrega esperada

Um repositório no GitHub com o código + um README explicando como rodar — primeiro item do portfólio do aventureiro.
