## O que você vai construir

Um pipeline pequeno: coletar dados de uma fonte pública (API ou scraping leve), salvar em CSV, ler com pandas, gerar 3-5 estatísticas/insights e exportar um relatório em Markdown.

## Habilidades exercitadas

- `requests` + tratamento de status HTTP
- BeautifulSoup ou parsing de JSON
- Pandas: leitura, filtros, agregações
- Geração de saída legível (tabela Markdown, gráfico simples com matplotlib)
- Estrutura de pipeline em etapas

## Entrega esperada

Repositório com instruções de reprodução, dados de exemplo (anonimizados se necessário) e relatório final commitado.
