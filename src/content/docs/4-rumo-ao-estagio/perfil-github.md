## O que você vai aprender

- Username profissional (nome real é melhor; evite gamertag)
- Foto, bio, localização, links que importam
- README de perfil: o repositório especial com seu nome
- Pinned repos: escolher os 6 melhores trabalhos para destacar
- Frequência de commits — sem fingir, mas sem deixar morrer
