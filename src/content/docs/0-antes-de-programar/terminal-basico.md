## O que você vai aprender

- Diferença entre PowerShell, cmd e o terminal integrado do VS Code
- Comandos para navegar: `cd`, `ls` / `dir`, `pwd`, voltar com `cd ..`
- Comandos para criar pastas e arquivos: `mkdir`, `New-Item`
- Como rodar um arquivo Python: `python script.py`
- Como cancelar um programa travado (Ctrl+C) e limpar a tela (`cls`)
