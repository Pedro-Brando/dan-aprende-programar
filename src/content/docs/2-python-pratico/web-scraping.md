## O que você vai aprender

- Combinar `requests` com `beautifulsoup4` para baixar e parsear HTML
- Selecionar elementos com tag, classe e CSS selectors
- Extrair texto, links e atributos
- Lidar com páginas que carregam por JavaScript (limitação do approach simples)
- Ética e legalidade: `robots.txt`, rate limit, termos de uso
