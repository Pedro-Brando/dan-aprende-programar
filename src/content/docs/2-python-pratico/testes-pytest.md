## O que você vai aprender

- Por que testar código manualmente não escala
- Instalar `pytest` e rodar com `pytest` no terminal
- Escrever um teste: arquivo `test_*.py`, função `test_*`, `assert`
- Estruturar testes com `Arrange / Act / Assert`
- Introdução rápida a fixtures (`@pytest.fixture`) para preparar dados
