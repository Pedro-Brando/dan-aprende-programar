## O que você vai aprender

- Instalar `requests` no venv com `pip install requests`
- Fazer um GET e ler `response.status_code`, `response.json()`
- Enviar parâmetros de query e headers
- Lidar com erros: timeouts, 404, 500
- Brincar com APIs públicas (ex: ViaCEP, PokeAPI) sem precisar de chave
