## O que você vai aprender

- O que é JSON e por que está em todo lugar
- Mapear JSON ↔ dicionários e listas Python
- `json.loads`/`json.dumps` (string) e `json.load`/`json.dump` (arquivo)
- Salvar dados de um programa em arquivo `.json` para abrir na próxima execução
- Lidar com tipos que JSON não entende (datas, classes próprias)
